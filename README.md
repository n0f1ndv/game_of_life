## Game of Life
In this project I am trying to replicate Conway's Game of Life.

## Installation & Compliation
Get the repository:
``` bash
git clone https://github.com/n0f1ndv/game_of_life.git
```

Create build directories:
``` bash
mkdir build
cmake -S . -B build
```

Build the project:
``` bash
cmake --build build
```

Run the app, it will display available options:
``` bash
./build/app
```
